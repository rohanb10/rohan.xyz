I was frustrated with the inability to edit rides on Strava and I couldn't find a free tool on the internet to help me out. So naturally, I made one myself.

Snoozz
2020

I had too many tabs open so I made a web extension to help reduce that clutter. This webapp lets you snoozz tabs now and have them reopen later when you actually need them.

You can install the extension from the Chrome Extension Store or the Firefox Add-On Market and view the source code in this github repo.

  • Snooze tabs for later using a button in the toolbar with just two clicks.
  • Wake up tabs automatically at the pre-determined time.
  • Right click urls anywhere on the web to snooze a tab to open later when you have time to read it.
  • View a list of all your snoozed tabs, sorted by date and time and wake them up manually.
  • View a history of your woken up tabs and purge the history automatically after a user configured period of time.
  • Written in vanilla JS with no libraries - file size < 40kb.

Path Editor
2020

I was frustrated with the inability to edit rides on Strava and I couldn't find a free tool on the internet to help me out. So naturally, I made one myself.

It's live and free to use on this website. The webapp is open sourced and you can view all the code and sample files in this github repo.

  • Import rides and display them as overlays over Google Maps as a base layer.
  • Draw and drag new vertices to points that were not recorded.
  • Edit and move vertices on paths that may have been incorrectly placed.
  • Delete vertices that are superfluous to reduce size of path.
  • Draw and edit shapes to select vertices in bulk for deletion.
  • Export coordinates in several different formats including CSVs and polylines.
Snoozz
2020

I had a tab hoarding problem and this is how I solved it.

A WebExtension to snoozz your tabs and windows now, and have them wake up automatically later. It's completely open source and published everywhere for free.

The feature packed app is written in vanilla JS with a focus on performance, privacy and a low memory footprint. The whole package including all image assets is ˜ 90-100 KB, and it can be used completely offline. It has a lot of handy features, with more down the pipeline:

Visit Snoozz.me for more details, FAQ, changelogs and all that fun stuff.

  • Snooze individual tabs, selected tabs or full windows from the toolbar in just two clicks.
  • Right click on any link in your browser to open up a context menu to snooze tabs without even opening them.
  • Configure keyboard shortcuts scoped to the browser to snooze tabs instantly using your preferred nap times.
  • View all your sleeping tabs in the "Nap Room" where you can disturb them.
  • Lots of customisable settings - Choose Icon Packs, Dark Mode, Notifications etc.
  • Many more features in active development: cloud sync, edit snooze times, recurring naps.
